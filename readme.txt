== Installation ==

### Wordpress Admin
1. Search 'wpapper' plugin
2. Install and activate the plugin
3. Find the page through: Settings -> Wpapper

### Or ftp method
1. Open `wp-content/plugins` Folder
2. Put: `Folder: wpapper`
3. Activate `wpapper` Plugin
4. Go to `WP-Admin -> Settings -> WpApper` to configure the plugin.
5. Go to http://wpapper.com/ submit web info

For detailed instructions http://wpapper.com/
